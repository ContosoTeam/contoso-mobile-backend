# Contoso Mobile Backend

REST API backend for the Contoso mobile application, built with Java Spring Boot and MySQL.

## Features
- User registration and authentication
- Push notification management
- Content delivery API
- In-app purchase verification
- User profile management
- Activity logging

## Quick Start

```bash
mvn spring-boot:run
```

## Technology Stack
- Java 17
- Spring Boot 3.2
- Spring Data JPA
- MySQL 8.0
- Redis (session management)
- Firebase Cloud Messaging
